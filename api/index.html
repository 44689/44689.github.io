API文档

请求链接：

http://net.ccbtv.top/qr/api.php

请求方式：GET

url参数：

参数名 类型 内容 必要性 备注
qr str 转换的内容 必要 内容填包含http://或https://的二维码可直接跳转链接
download str yes 可选 如果要这个参数，则会跳出下载二维码，如果没有，就正常显示二维码

返回内容：

返回类型 返回内容 备注
img 返回一个二维码图片 如果没选择download参数，则返回这条内容
att 返回一个下载链接 如果选择了download参数，则返回这条内容
